# 전역 경제 제재 검색 서비스 소스 코드

이 디렉토리는 경제 제재 검색 웹 애플리케이션의 소스 코드를 포함합니다.

## 프로젝트 구조

```
src/
├── assets/         # 이미지, 데이터 파일 등 정적 자산
├── css/            # 스타일시트 파일
│   ├── base/       # 기본 스타일 (변수, 리셋 등)
│   ├── components/ # 컴포넌트별 스타일
│   ├── layout/     # 레이아웃 관련 스타일
│   ├── themes/     # 테마 스타일
│   └── utilities/  # 유틸리티 클래스
├── js/             # JavaScript 파일
│   ├── animations/ # 애니메이션 관련 스크립트
│   ├── components/ # UI 컴포넌트 스크립트
│   ├── services/   # API 및 서비스 스크립트
│   ├── utils/      # 유틸리티 함수
│   └── app.js      # 메인 애플리케이션 파일
└── index.html      # 메인 HTML 파일
```

## 컴포넌트 구조

애플리케이션은 다음과 같은 주요 컴포넌트로 구성됩니다:

- **alerts.js**: 알림 및 토스트 메시지 컴포넌트
- **detail.js**: 제재 대상 상세 정보 표시 컴포넌트
- **filters.js**: 검색 필터 컴포넌트
- **search.js**: 검색 기능 컴포넌트
- **user-auth.js**: 사용자 인증 컴포넌트

## 개발 가이드라인

### 스타일 작업

- 새로운 스타일은 적절한 카테고리 폴더에 배치하세요.
- 컴포넌트별 스타일은 `css/components/` 디렉토리에 배치하세요.
- 전역 변수는 `css/base/variables.css`에 정의하세요.

### JavaScript 작업

- 컴포넌트 기능은 해당 컴포넌트 파일에 캡슐화하세요.
- 유틸리티 함수는 `js/utils/` 디렉토리에 배치하세요.
- API 호출 및 데이터 처리는 `js/services/` 디렉토리에 배치하세요.

### 애니메이션

- 애니메이션 관련 코드는 `js/animations/` 디렉토리에 있습니다.
- 새로운 애니메이션은 적절한 모듈에 추가하고 `index.js`에서 통합하세요.

## 빌드 프로세스

프로젝트를 빌드하려면:

```
# 프로젝트 루트 디렉토리에서 빌드 스크립트 실행
./build.ps1
```

빌드된 파일은 `dist/` 디렉토리에 생성됩니다.
